## 一、项目概述

星火联盟是一个以AI为特色入口,深度融合社交、生活服务与个人管理的校园超级App。目标用户为大学生群体,未来将扩展到青年群体(学生、职场人士、自由职业者等)。

**核心定位**: 通过AI技术提供智能化服务,同时构建完整的校园生态系统。

## 二、核心功能模块详解

### 2.1 AI模式(流量入口与特色)

#### 功能定位
AI模式是星火联盟的流量入口和核心特色,通过集成多个AI模型为用户提供智能服务。

#### 核心功能
1. **多模型集成**
   - 支持DeepSeek、豆包等多个AI模型
   - 服务端代理模式,降低用户使用门槛
   - 用户可选择自填API Key,降低平台成本

2. **智能对话**
   - 悬浮窗设计,随时可用
   - 历史记录侧边栏,方便查看
   - 独立对话记录,不同步其他平台

3. **混合模式**
   - 平台提供基础额度供用户试用
   - 用户可绑定自己的API Key
   - App代理请求或前端直连,灵活选择

#### 技术方案
- **服务端代理**: 使用Edge Function代理AI请求
- **用户自持**: 支持用户填写自己的API Key
- **数据隔离**: App内独立存储对话记录,确保安全
- **授权机制**: 用户必须明确授权使用AI服务

#### 与其他模块的联系
- **智能日程**: AI可以帮助用户规划日程
- **星火规划**: AI提供任务规划建议
- **校园购物**: AI提供商品推荐和建议
- **学习资源**: AI辅助学习答疑

### 2.2 社交伴侣(核心粘性功能)

#### 功能定位
社交伴侣是用户粘性最强的功能,通过社交互动连接用户。

#### 核心功能
1. **人格化AI助手**
   - AI"星火助手"提供陪伴式对话
   - 支持混合群聊(三人+一AI)
   - 智能回复和情感陪伴

2. **真人社交**
   - 通过ID/二维码添加好友
   - 一对一私聊
   - 群组聊天

3. **动态分享**
   - 类似朋友圈的动态发布
   - 图片、文字、视频分享
   - 点赞、评论、收藏互动

4. **星火域**
   - 校园式朋友圈
   - 动态可见性控制
   - 社交互动激励

#### 技术方案
- **实时通信**: 使用WebSocket实现实时消息
- **消息存储**: Supabase Realtime存储消息
- **图片存储**: Supabase Storage存储图片
- **关系管理**: 好友关系、群组关系数据库管理

#### 与其他模块的联系
- **AI模式**: AI参与群聊,提供智能回复
- **校园墙**: 动态可以分享到校园墙
- **星火资讯**: 资讯可以分享到星火域
- **星火人才**: 职场社交和招聘信息

### 3.5 星火人才双向匹配(Spark Talent)

#### 功能定位
颠覆传统招聘模式,从"企业发布→学生投递"的单向被动模式,转变为"青年展示能力→企业主动寻访"的双向主动模式。

#### 核心逻辑
**传统模式**: 企业发岗位 → 学生被动投简历(低效、信息不对称)  
**Spark模式**: 青年/企业均可主动发起 → AI智能双向匹配 → 高效精准连接

#### 青年侧功能
1. **能力名片**
   - 擅长领域标签(编程/设计/运营/写作/数据分析/视频剪辑等)
   - 作品集展示(图片/项目链接/视频演示/GitHub仓库)
   - 项目经历(可挂载校园墙发布的项目动态)
   - 可接受合作类型(实习/兼职/项目外包/全职)
   - 期望薪资范围(可选填)
   - 可见性控制(公开/仅企业可见/仅好友可见)

2. **AI能力标签**
   - 自动提取与推荐
   - 智能匹配岗位

3. **邀请管理**
   - 接收企业邀请
   - 接受/拒绝/稍后处理
   - 全流程投递状态追踪

4. **匿名保护**
   - 隐藏真实姓名
   - 企业感兴趣后再解锁

#### 企业侧功能
1. **企业认证**
   - 营业执照/官方邮箱/工商信息三重验证
   - 防钓鱼防欺诈

2. **岗位发布**
   - 学生可主动投递,双向通道

3. **人才搜索**
   - 按技能标签/院校/城市/合作类型/匹配度筛选
   - AI智能推荐候选人(语义匹配度打分)

4. **邀约管理**
   - 向目标青年发送邀约
   - 频率限制防骚扰
   - 候选人沟通进度管理(看板式流程追踪)

#### AI匹配机制
- **语义向量匹配**: 企业JD描述 ↔ 青年能力名片(cosine相似度)
- **匹配度分值**: 0-100%可视化展示
- **多维度权重**: 技能匹配70% + 院校背景15% + 活跃度15%
- **持续优化**: 基于历史行为推荐(协同过滤)
- **黑名单机制**: 保护青年免受骚扰

#### 安全与合规
- 企业认证后才可主动联系
- 单日联系频率上限
- 举报投诉快速响应
- 可选接入学信网核验
- 所有沟通记录留存,纠纷可追溯

#### 商业模式
- 青年侧永久免费
- 企业侧: 基础版免费(每月限10次联系)/ 专业版付费(无限联系+AI推荐+数据报告)/ 旗舰版(置顶展示+专属客服+校园宣讲联动)
- 猎头/HR服务平台撮合佣金(未来)

#### 与其他模块的联系
- **校园墙**: 项目经历挂载
- **星火资讯**: 就业资讯聚合
- **社交伴侣**: 职场社交
- **AI模式**: AI智能匹配

### 3.6 星火资讯(热点爬虫聚合)

#### 功能定位
打破信息茧房,让校园青年不脱离社会脉搏。通过爬虫聚合全网热点,结合用户兴趣实现个性化推送。

#### 信息源策略
1. **新闻媒体**: 新华社、人民日报、澎湃新闻、36氪、虎嗅
2. **社交平台**: 微博热搜、知乎热榜、B站热门、抖音热点
3. **垂直领域**: 少数派(科技)、雪球(财经)、天眼查(商业动态)
4. **官方渠道**: 教育部通知、地方政府公告
5. **国际资讯**: BBC中文、路透社中文
6. **就业市场**: BOSS直聘趋势、智联招聘薪资报告

#### 爬虫架构
- **调度层**: 定时任务(热榜每15分钟抓取,深度内容每小时更新)
- **去重层**: URL指纹哈希 + 内容语义向量去重
- **清洗层**: 正文抽取、广告过滤、敏感内容过滤
- **存储层**: 原文快照 + 结构化摘要入库
- **合规层**: 遵守robots.txt,保留原文链接,不侵权存储

#### 前端功能模块
1. **今日热榜**
   - 多源融合排序
   - 资讯卡片: 标题 + 来源 + 热度 + AI摘要 + 情感标签
   - 实时热度趋势: 上升/下降/新入榜/持续发酵

2. **分类频道**
   - 一级分类: 科技与数字/财经商业/校园学术/社会民生/娱乐文化/国际视野/本地资讯
   - 二级标签叠加筛选
   - 时间范围筛选
   - 热度筛选

3. **为你推荐**
   - 用户主动设置兴趣标签
   - 隐式兴趣识别(基于阅读行为)
   - "不感兴趣"标记
   - 信息茧房破防机制

4. **深度阅读**
   - 原文快照展示
   - AI一键摘要
   - AI多角度解读
   - 关联联动入口
   - 相关资讯推荐

5. **社交联动**
   - 一键分享到校园墙
   - 收藏入个人知识库
   - 资讯评论区
   - 与Spark Talent联动

6. **每日简报**
   - "今日必看"定时推送
   - 用户可自定义推送时间和数量
   - 关键词订阅
   - 周报/月报

#### 内容安全体系
- 三级过滤: 爬取时 → 入库前 → 展示前
- 敏感词过滤库动态更新
- 用户举报机制
- 争议内容标注
- 涉政内容从严处理

#### 技术方案
- **爬虫框架**: Python + Scrapy/Playwright
- **任务调度**: Celery + Redis
- **存储**: PostgreSQL + Elasticsearch
- **AI摘要**: DeepSeek API
- **向量存储**: pgvector
- **缓存**: Redis
- **部署**: 独立微服务

#### 商业模式
- 基础资讯流: 永久免费
- 付费功能: 无限历史存档/专项深度报告/关键词监控
- B端服务: 行业舆情监控订阅

#### 与其他模块的联系
- **校园墙**: 资讯分享和讨论
- **社交伴侣**: 资讯分享到星火域
- **星火人才**: 财经/就业类资讯下方展示岗位推荐
- **AI模式**: AI摘要和多角度解读

### 3.7 AI选课助手

#### 功能定位
利用AI技术帮助学生智能选课,避免踩坑,提高选课质量。

#### 核心功能
1. **课程评价系统**
   - 学生真实评价(匿名)
   - 历年选课数据统计
   - 成绩分布分析
   - 教师教学风格标签
   - 评价维度: 课程难度、教学质量、实用性、作业量、考试形式、给分情况

2. **AI智能推荐**
   - 用户输入: 专业/年级、兴趣方向、学习目标、时间偏好、难度接受度
   - AI分析: 匹配课程评价数据、分析历年趋势、考虑时间冲突、平衡难度
   - 输出: 推荐课程列表、详细分析、选课建议、备选方案

3. **学长学姐经验分享**
   - 文字经验分享、视频课程测评
   - 选课避坑指南、学习方法建议
   - 提问互动、点赞收藏、评论私信

4. **课程对比功能**
   - 同一课程不同教师对比
   - 同类课程横向对比
   - 历年数据趋势对比
   - 可视化展示: 雷达图、柱状图、折线图、词云图

#### 与其他模块的联系
- **智能日程**: 选课后自动添加到日程
- **学长推荐系统**: 经验分享
- **AI模式**: AI智能推荐课程
- **学习资源中心**: 课程资料分享

### 3.8 学长推荐系统

#### 功能定位
建立学长学姐与学弟学妹的连接桥梁,实现经验传承和互助成长。

#### 核心功能
1. **学长学姐认证**
   - 认证方式: 学生证、成绩单、获奖证书、活动经历
   - 认证等级: 普通学长、优秀学长、金牌学长

2. **经验分享平台**
   - 学习经验: 专业课程方法、考试技巧、笔记资料、学习规划
   - 生活经验: 校园指南、美食推荐、宿舍技巧、周边攻略

   - 考研留学: 备考经验、院校选择、申请指南、奖学金技巧

   - 咨询类型: 文字(免费)、语音(付费)、视频(付费)、线下(付费)
   - 激励机制: 学长获得积分/报酬、优质回答奖励、认证提升

4. **经验内容库**
   - 内容分类: 按专业、年级、主题、热度
   - 内容形式: 图文教程、视频讲解、直播分享、问答合集
   - 智能推荐: 根据专业、年级、兴趣个性化推荐

## 八、总结

星火联盟是一个功能丰富、模块化的校园超级App,通过AI技术提供智能化服务,同时构建完整的校园生态系统。

**核心特点**:
1. **AI驱动**: AI作为核心特色和流量入口
2. **社交连接**: 社交功能贯穿整个平台
3. **校园场景**: 针对校园场景深度优化
4. **游戏化成长**: 通过游戏化激励用户持续使用
5. **模块化设计**: 各模块相对独立又相互关联

**模块关联**:
- AI模式是核心,渗透到所有模块
- 社交伴侣是连接器,连接所有社交功能
- 校园购物是变现核心
- 智能日程和星火规划是个人管理核心
- 其他扩展功能丰富平台生态

**技术共享**:
- 所有模块共享数据库、存储、认证、实时通信等基础设施
- 数据在模块间流转,形成完整的生态闭环
- 服务集成降低开发成本,提高开发效率

**商业模式**:
- 免费功能引流
- 付费功能变现
- B端服务规模化盈利

这个设计确保了星火联盟能够成为一个功能完整、体验优秀、可持续发展的校园超级App。
