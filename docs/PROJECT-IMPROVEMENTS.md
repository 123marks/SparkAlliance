## 一、项目核心定位

### 1.1 项目概述

**项目名称**: 星火校园 (Spark Alliance)  
**核心定位**: 以AI为特色入口，深度融合社交、生活服务与个人管理的校园超级App  
**目标用户**: 大学生 → 青年群体（学生、职场人士、自由职业者等）  
**发展路径**: 校园版 → 社区版 → 国外版

### 1.2 商业模式

- **前期**: 免费AI和工具功能吸引用户（引流）
- **中期**: 二手交易撮合费和增值服务（初步盈利）
- **后期**: B端商家、会员体系、高利润商品（规模化盈利）

## 二、七大功能模块

### 功能一：AI模式（流量入口与特色）

**核心功能：**
- 集成多模型（DeepSeek、豆包等）
- 本地授权方案（不使用免费API）
- 悬浮窗设计
- 历史记录侧边栏

**技术方案：**
- ⚠️ **重要决策**: 采用本地授权方案
- 用户在本地登录DeepSeek/豆包等模型
- App检测本地登录状态并获取授权
- App内独立对话记录，不同步其他平台

**注意事项：**
- 需要研究各平台的授权机制
- 确保数据隔离和安全性
- 用户必须明确授权

### 3.5 星火人才双向匹配（Spark Talent）

**核心定位：** 颠覆传统招聘模式，青年主动展示能力，企业按需寻访，双向匹配。

**青年侧：**
- 创建能力名片（擅长标签/作品集/项目经历/合作类型）
- 可见性控制 + 匿名保护选项
- AI自动提取能力标签
- 企业邀请管理 + 全程投递状态追踪

**企业侧（需三重认证）：**
- 发布岗位 + 主动搜索人才库
- AI语义推荐候选人（匹配度打分）
- 候选人沟通看板管理

**匹配机制：** 语义向量匹配 + 协同过滤优化，匹配度0-100%可视化

**商业模式：** 青年永久免费；企业基础版免费（每月10次联系）/ 专业版付费 / 旗舰版（置顶+校园宣讲）

**开发优先级：** 第三阶段（5-6月）

### 3.6 星火资讯（热点爬虫聚合）

**核心定位：** 打破信息茧房，聚合全网热点，个性化推送，AI辅助深度理解。

**信息源：** 新华社/澎湃/36氪/虎嗅 + 微博/知乎/B站热榜 + 教育部公告 + 就业市场数据

**六大功能模块：**
- 今日热榜：多源融合 + 情感标签 + 实时热度趋势
- 分类频道：7大一级分类 + 多标签叠加筛选
- 个性化推荐：兴趣标签 + 隐式行为识别 + 信息茧房破防机制
- 深度阅读：AI三句摘要 + 正/反/中立三角度解读
- 社交联动：一键分享校园墙 + 与Spark Talent打通
- 每日简报：定时推送 + 关键词订阅

**技术：** Scrapy/Playwright + Celery + ES + pgvector，独立微服务部署

**开发优先级：** 第三阶段（5-6月）

## 四、技术选型

### 4.1 数据库方案

**决策：**
- **前期**: 使用Supabase
- **后期**: 可迁移到其他数据库

**原因：**
- Supabase提供完整后端服务
- 快速开发，降低初期成本
- PostgreSQL兼容性好

**注意事项：**
- ⚠️ 使用标准SQL，避免Supabase特有功能
- ⚠️ 预留数据导出接口
- ⚠️ 设计数据迁移方案

---

### 4.2 前端框架

**选项：**
- React Native
- Flutter

**待确定：**
- 根据团队技术栈决定
- 考虑性能需求

---

### 4.3 后端技术

**推荐：**
- Node.js + Express / NestJS
- 或 Go

---

### 4.4 AI模型接入

**方案：本地授权方案**

**流程：**
1. 检测用户本地是否登录DeepSeek/豆包等
2. 请求用户授权App使用该模型
3. 获取授权Token
4. 在App内调用模型API
5. 独立存储对话记录

**优势：**
- 不需要免费API的复杂接入
- 用户体验更好
- 数据更安全

**挑战：**
- 需要各平台支持授权
- 技术实现需要研究

## 十、新增功能详细规划

### 10.1 AI选课助手

**功能定位：** 利用AI技术帮助学生智能选课，避免踩坑，提高选课质量。

**核心功能：**
1. **课程评价系统**
   - 学生真实评价（匿名）
   - 历年选课数据统计
   - 成绩分布分析
   - 教师教学风格标签
   - 评价维度：课程难度、教学质量、实用性、作业量、考试形式、给分情况

2. **AI智能推荐**
   - 用户输入：专业/年级、兴趣方向、学习目标、时间偏好、难度接受度
   - AI分析：匹配课程评价数据、分析历年趋势、考虑时间冲突、平衡难度
   - 输出：推荐课程列表、详细分析、选课建议、备选方案

3. **学长学姐经验分享**
   - 文字经验分享、视频课程测评
   - 选课避坑指南、学习方法建议
   - 提问互动、点赞收藏、评论私信

4. **课程对比功能**
   - 同一课程不同教师对比
   - 同类课程横向对比
   - 历年数据趋势对比
   - 可视化展示：雷达图、柱状图、折线图、词云图

**开发优先级：** 第二阶段（4-5月），基础评价系统优先

---

### 10.2 学长推荐系统

**功能定位：** 建立学长学姐与学弟学妹的连接桥梁，实现经验传承和互助成长。

**核心功能：**
1. **学长学姐认证**
   - 认证方式：学生证、成绩单、获奖证书、活动经历
   - 认证等级：普通学长、优秀学长、金牌学长

2. **经验分享平台**
   - 学习经验：专业课程方法、考试技巧、笔记资料、学习规划
   - 生活经验：校园指南、美食推荐、宿舍技巧、周边攻略
   - 职业发展：实习求职、简历面试、行业认知、职业规划
   - 考研留学：备考经验、院校选择、申请指南、奖学金技巧

3. **一对一咨询**
   - 咨询流程：学生提问 → AI匹配学长 → 学长接单 → 在线咨询 → 评价反馈
   - 咨询类型：文字（免费）、语音（付费）、视频（付费）、线下（付费）
   - 激励机制：学长获得积分/报酬、优质回答奖励、认证提升

4. **经验内容库**
   - 内容分类：按专业、年级、主题、热度
   - 内容形式：图文教程、视频讲解、直播分享、问答合集
   - 智能推荐：根据专业、年级、兴趣个性化推荐

**开发优先级：** 第二阶段（4-5月），基础分享平台优先

---

### 10.3 AI学习助手

**功能定位：** 为学生提供全方位的AI学习辅助，提高学习效率。

**核心功能：**
1. **智能答疑**
   - 24小时在线答疑
   - 支持图片识别（题目拍照）、语音提问
   - 分步骤讲解、举一反三推荐
   - 学科覆盖：数学、英语、专业课、通识课

2. **学习计划制定**
   - AI分析：学习目标、可用时间、学习基础、学习风格
   - 输出方案：每日计划、阶段目标、复习提醒、进度追踪

3. **知识点梳理**
   - 自动生成知识图谱
   - 重点难点标注、易错点提醒
   - 关联知识推荐

4. **模拟考试**
   - AI生成模拟题、自动批改评分
   - 错题分析、薄弱点定位
   - 针对性练习

**开发优先级：** 第一阶段（MVP），基础答疑功能优先

---

### 10.4 学习资源中心

**功能定位：** 整合优质学习资源，打造一站式学习平台。

**核心功能：**
1. **资源分类**
   - 课程资料：PPT、笔记、作业
   - 考试资料：真题、模拟题、答案
   - 学习工具：软件、网站、APP
   - 视频教程：网课、公开课、讲座
   - 电子书籍：教材、参考书、课外书

2. **资源上传与分享**
   - 激励机制：上传获积分、优质获奖励、分享获认可、贡献榜单展示
   - 质量控制：AI审核、用户举报、评分评价、定期清理

3. **资源搜索**
   - 搜索功能：关键词、分类筛选、标签搜索、智能推荐
   - 搜索优化：语义理解、模糊匹配、相关推荐、历史记录

**开发优先级：** 第三阶段（5-6月），基础资源平台优先
