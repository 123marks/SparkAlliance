## 🎓 AI选课助手

### 功能定位
利用AI技术帮助学生智能选课，避免踩坑，提高选课质量。

### 核心功能

#### 1. 课程评价系统
**数据来源：**
- 学生真实评价（匿名）
- 历年选课数据
- 成绩分布统计
- 教师教学风格标签

**评价维度：**
- 课程难度（1-5星）
- 教师教学质量（1-5星）
- 课程实用性（1-5星）
- 作业量（少/中/多）
- 考试形式（开卷/闭卷/论文/项目）
- 给分情况（严格/正常/宽松）

#### 2. AI智能推荐
**推荐逻辑：**
```
用户输入：
- 专业/年级
- 兴趣方向
- 学习目标（考研/就业/兴趣）
- 时间偏好（早课/晚课/无偏好）
- 难度接受度

AI分析：
- 匹配课程评价数据
- 分析历年选课趋势
- 考虑时间冲突
- 平衡课程难度
- 推荐最优选课方案
```

**推荐结果：**
- 推荐课程列表（按匹配度排序）
- 每门课程的详细分析
- 选课建议和注意事项
- 备选方案（如推荐课程已满）

#### 3. 学长学姐经验分享
**内容形式：**
- 文字经验分享
- 视频课程测评
- 选课避坑指南
- 学习方法建议

**互动功能：**
- 提问功能（向学长学姐提问）
- 点赞和收藏
- 评论互动
- 私信咨询

#### 4. 课程对比功能
**对比维度：**
- 同一课程不同教师对比
- 同类课程横向对比
- 历年数据趋势对比
- 就业导向对比

**可视化展示：**
- 雷达图（多维度评分）
- 柱状图（历年选课人数）
- 折线图（成绩分布）
- 词云图（学生评价关键词）

## 👨‍🎓 学长推荐系统

### 功能定位
建立学长学姐与学弟学妹的连接桥梁，实现经验传承和互助成长。

### 核心功能

#### 1. 学长学姐认证
**认证方式：**
- 学生证认证
- 成绩单认证（优秀学生）
- 获奖证书认证
- 活动经历认证

**认证等级：**
- 普通学长（基础认证）
- 优秀学长（成绩优异）
- 金牌学长（综合优秀+活跃分享）

#### 2. 经验分享平台
**分享内容：**
- **学习经验**
  - 专业课程学习方法
  - 考试备考技巧
  - 笔记和资料分享
  - 学习规划建议

- **生活经验**
  - 校园生活指南
  - 食堂美食推荐
  - 宿舍生活技巧
  - 周边玩乐攻略

- **职业发展**
  - 实习求职经验
  - 简历面试技巧
  - 行业认知分享
  - 职业规划建议

- **考研留学**
  - 考研备考经验
  - 院校选择建议
  - 留学申请指南
  - 奖学金申请技巧

#### 3. 一对一咨询
**咨询流程：**
```
学生提问 → AI智能匹配学长 → 学长接单 → 在线咨询 → 评价反馈
```

**咨询类型：**
- 文字咨询（免费）
- 语音咨询（付费）
- 视频咨询（付费）
- 线下见面（付费）

**激励机制：**
- 学长获得积分/报酬
- 优质回答获得奖励
- 活跃学长获得认证提升
- 学生获得满意答案

#### 4. 经验内容库
**内容分类：**
- 按专业分类
- 按年级分类
- 按主题分类
- 按热度排序

**内容形式：**
- 图文教程
- 视频讲解
- 直播分享
- 问答合集

**智能推荐：**
- 根据学生专业推荐相关经验
- 根据学生年级推荐适合内容
- 根据学生兴趣个性化推荐
- 根据热门趋势推荐优质内容

## 🤖 AI学习助手

### 功能定位
为学生提供全方位的AI学习辅助，提高学习效率。

### 核心功能

- 复习提醒
- 进度追踪

#### 3. 知识点梳理
**功能：**
- 自动生成知识图谱
- 重点难点标注
- 易错点提醒
- 关联知识推荐

#### 4. 模拟考试
**功能：**
- AI生成模拟题
- 自动批改评分
- 错题分析

## 📊 数据模型

### 课程评价表
```sql
CREATE TABLE course_reviews (
  id UUID PRIMARY KEY,
  course_id UUID,
  user_id UUID,
  difficulty_rating INT,
  teaching_quality INT,
  practicality_rating INT,
  workload_level VARCHAR(20),
  exam_type VARCHAR(20),
  grading_style VARCHAR(20),
  comment TEXT,
  created_at TIMESTAMP
);
```

### 学长认证表
```sql
CREATE TABLE senior_certifications (
  id UUID PRIMARY KEY,
  user_id UUID,
  certification_type VARCHAR(50),
  certification_level VARCHAR(50),
  verified BOOLEAN,
  verified_at TIMESTAMP,
  created_at TIMESTAMP
);
```

### 经验分享表
```sql
CREATE TABLE experience_shares (
  id UUID PRIMARY KEY,
  author_id UUID,
  title VARCHAR(200),
  content TEXT,
  category VARCHAR(50),
  tags TEXT[],
  likes_count INT,
  views_count INT,
  created_at TIMESTAMP
);
```
