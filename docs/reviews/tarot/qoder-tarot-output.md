## 功能定位建议

### 核心差异化定位：校园场景化AI玄学伴侣

**为什么叫"卡罗牌"而不是"塔罗牌"？**

这是刻意为之的品牌策略。"卡罗"取自"Campus Tarot"的谐音，既保留了塔罗的神秘感，又暗示了校园专属属性。建议围绕这个命名做三层差异化：

| 维度 | 传统塔罗App | 星火卡罗牌 |
|------|-------------|-----------|
| **内容调性** | 严肃神秘、宗教感 | 轻松有梗、校园化语言 |
| **解读方式** | 固定牌义+简单随机 | AI深度个性化+场景绑定 |
| **使用场景** | 泛娱乐 | 考试周/恋爱/社团/宿舍生活 |
| **社交属性** | 弱 | 强（校园墙联动、好友互测） |

**核心用户场景优先级（按真实需求排序）：**

1. **考试前心理暗示**（刚需场景）
   - "期末考试能过吗" "四六级会怎样"
   - 结合日程模块，抽完自动添加复习提醒
   
2. **感情困惑解读**（高频场景）
   - "TA对我有意思吗" "要不要表白"
   - 大学生恋爱焦虑的出口
   
3. **每日心理能量补给**（习惯养成）
   - 每日一抽，获得今日"星火指引"
   - 类似"每日一签"但更有深度
   
4. **宿舍/朋友间的社交游戏**（传播场景）
   - "帮室友抽一张" "我们三个人测一下"
   - 天然社交货币

**功能边界建议：**

采用**轻入口 + 深体验**模式：
- **轻入口**：首页悬浮星火图标、AI助手快捷入口
- **深体验**：点击后进入独立沉浸式卡罗牌空间
- **渗透入口**：个人中心展示"守护星火牌"、日程模块"今日运势"小组件

## 卡牌体系设计方案

### 推荐方案：双层卡牌体系（传统78张 + 校园专属12张）

**第一层：简化版传统塔罗（22张大阿尔卡那）**

保留塔罗最核心的22张大牌，降低学习成本的同时保持神秘感：

| 牌名 | 校园化解读角度 |

| 皇帝 | 领导力、社团管理 |
| 恋人 | 感情抉择、室友关系 |
| 战车 | 考试冲刺、竞赛突破 |
| 力量 | 坚持自律、健身打卡 |
| 隐者 | 独处思考、考研备战 |
| 命运之轮 | 运气转变、选课结果 |
| 正义 | 公平对待、奖学金评选 |
| 倒吊人 | 换个角度、gap思考 |
| 死神 | 结束与新生、毕业季 |
| 节制 | 平衡生活、时间管理 |
| 恶魔 | 诱惑陷阱、游戏上瘾 |

| 太阳 | 好运降临、顺利通过 |
| 审判 | 重新评估、转专业 |
| 世界 | 圆满完成、毕业达成 |

**第二层：星火专属校园牌（12张）**

这是差异化核心，完全原创设计：

| 牌名 | 视觉意象 | 核心含义 |
|------|----------|----------|
| **开学星火** | 燃烧的新书本 | 新开始的能量与期待 |
| **期末烈焰** | 图书馆燃烧的蜡烛 | 考试冲刺的决心与压力 |
| **社团星辰** | 多彩的星座连线 | 社交扩展、找到组织 |
| **暗恋微光** | 若隐若现的萤火虫 | 隐秘的心动与期待 |
| **宿舍星空** | 四张床铺上的星星 | 室友关系、生活和谐 |
| **食堂炊烟** | 热腾腾的饭菜蒸汽 | 基础生活、能量补给 |
| **图书馆灯塔** | 书堆中的灯光 | 知识探索、学业指引 |
| **实习之门** | 半开的职场大门 | 社会过渡、职业探索 |
| **假期银河** | 自由流动的星河 | 放松休整、探索自我 |
| **毕业极光** | 绚烂的极光 | 告别与新旅程 |
| **星火守护者** | 手捧火焰的人影 | 贵人相助、获得支持 |
| **迷雾考验** | 浓雾中的路口 | 困难挑战、需要突破 |

**正位/逆位设计：简化处理**

- **大阿尔卡那**：保留正逆位，但用"正向能量 / 提醒能量"替代晦涩术语
- **星火专属牌**：不设逆位，用"强/弱"能量表示程度
- 解读时AI会自动解释含义，无需用户理解塔罗知识

**MVP最小可行牌数：22张**

仅使用22张大阿尔卡那即可跑通核心体验，校园专属牌作为1.1版本迭代内容。

## AI解读机制设计

### 三层个性化解读架构

```
┌─────────────────────────────────────────────────┐
│                用户输入层                         │
│  问题文本 + 情绪标签 + 时间上下文 + 用户画像        │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│              AI语义理解层                         │
│  问题分类 → 意图识别 → 情感分析 → 场景匹配         │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│              个性化解读生成层                      │
│  牌义基础 × 问题关联 × 用户档案 × 风格适配         │
└─────────────────────────────────────────────────┘
```

### 个性化维度

| 维度 | 数据来源 | 解读影响 |
|------|----------|----------|
| **问题内容** | 用户输入 | 解读方向聚焦 |
| **情绪状态** | 可选标签（开心/焦虑/迷茫/期待） | 语气温度调整 |
| **星座/MBTI** | 用户档案（可选填） | 性格化解读角度 |
| **抽牌时间** | 系统自动 | 时段特殊含义（凌晨/期末周/开学季） |
| **历史记录** | 累计数据 | 避免重复、追踪趋势 |

### 三种解读风格（用户可切换）

**1. 学术理性风（默认）**
> "这张「战车」牌出现在你询问四六级的时刻，象征着强大的推进力与必胜决心。从能量角度看，你目前处于冲刺的最佳状态。建议：保持节奏，不要在最后一周突然改变复习策略。"

**2. 轻松调侃风**
> "哟，抽到「战车」了！这波稳了兄弟。四六级算什么，就你这气势，六级都得给你让路。不过别飘，该背的单词还是得背，玄学只负责加Buff，伤害还得你自己打。"

**3. 诗意文学风**
> "战车驰骋于星火之路，你便是那执缰者。此刻的你，如同黎明前的天际，即将迎来第一缕曙光。四六级不过是旅途中的一座小山，翻越它，你将看见更辽阔的风景。"

### 大学生高频问题预设模板

**学业类（40%预估占比）**
- 这次考试能过吗？
- 选A老师还是B老师的课？
- 要不要考研/考公？
- 毕设能顺利完成吗？
- 这学期能拿奖学金吗？

**感情类（35%预估占比）**
- TA对我有意思吗？
- 我应该表白吗？
- 这段关系还要继续吗？
- 我的正缘什么时候出现？
- 异地恋能坚持下去吗？

**生活/社交类（15%预估占比）**
- 要不要加入这个社团？
- 室友关系怎么处理？
- 这份实习该不该去？
- 今天运势怎么样？
- 我的财运如何？

**自我探索类（10%预估占比）**
- 我适合什么职业？
- 最近为什么状态不好？
- 我该怎么突破瓶颈？

## 社交传播玩法

### 核心传播机制：星火卡罗牌分享卡

**分享卡设计要素：**

```
┌────────────────────────────────────┐
│  ✨ 我在星火校园抽到了              │
│                                    │
│         [卡牌精美图片]               │
│           「战  车」                │
│                                    │
│  "胜利在望，保持冲劲"               │
│                                    │
│  ─────────────────────             │
│  问题：四六级能过吗？               │
│  时间：2026.03.25 期末周            │
│  ─────────────────────             │
│                                    │
│  [扫码同款占卜]  [星火校园]          │
└────────────────────────────────────┘
```

### 三大社交玩法

**1. 一键分享到校园墙**

- 分享时自动生成精美卡片
- 可选择隐藏具体问题（只显示"某同学问了一个关于学业的问题"）
- 校园墙上形成"今日星火牌"话题聚合
- 其他用户可点赞、评论、"同求好运"

**2. 每日同款机制（全校共振）**

- 每天零点生成"今日星火牌"（所有用户第一次抽牌必出这张）
- 全校看到同一张牌，但AI解读因人而异
- 形成话题："今天全校都是「塔」，大家都在渡劫吗？"
- 期末周特供：连续7天「战车」「力量」「太阳」等正向牌

**3. 好友互抽玩法**

- **帮TA抽一张**：为好友抽牌，解读发送到对方消息
- **双人合牌**：两人各抽一张，AI解读两人关系
- **宿舍四人牌阵**：4人组队，生成宿舍运势全景图
- **CP占卜**：输入两人信息，AI分析缘分指数

### 传播激励机制

| 行为 | 获得 |
|------|------|
| 首次分享到校园墙 | 解锁「星火传播者」成就 |
| 分享被10人点赞 | 获得1次额外抽牌机会 |
| 邀请好友首次占卜 | 双方各得3次抽牌机会 |
| 连续7天占卜 | 解锁「星火信徒」称号 |

| 风格切换 | P2 | 三种解读风格 |
| 好友互抽 | P2 | 社交功能 |

### 数据表设计（Supabase）

```sql
-- 用户占卜档案
CREATE TABLE user_tarot_profile (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    zodiac VARCHAR(20),           -- 星座
    mbti VARCHAR(4),              -- MBTI
    preferred_style VARCHAR(20) DEFAULT 'academic', -- 解读风格偏好
    total_draws INT DEFAULT 0,    -- 总抽牌次数
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 抽牌记录
CREATE TABLE tarot_draw_records (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    card_id INT NOT NULL,         -- 牌ID（1-22）
    card_name VARCHAR(50),        -- 牌名
    is_reversed BOOLEAN DEFAULT FALSE, -- 是否逆位
    question TEXT,                -- 用户问题
    question_category VARCHAR(20), -- 问题分类
    emotion_tag VARCHAR(20),      -- 情绪标签
    ai_interpretation TEXT,       -- AI解读内容
    interpretation_style VARCHAR(20), -- 使用的解读风格
    is_daily_card BOOLEAN DEFAULT FALSE, -- 是否今日同款
    shared_to_wall BOOLEAN DEFAULT FALSE, -- 是否分享到校园墙
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 今日星火牌（每日刷新）
CREATE TABLE daily_tarot_card (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    card_id INT NOT NULL,
    card_name VARCHAR(50),
    date DATE UNIQUE NOT NULL,
    theme_message TEXT,           -- 今日主题语
    draw_count INT DEFAULT 0,     -- 当日抽取人数
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 卡牌基础数据
CREATE TABLE tarot_cards (
    id SERIAL PRIMARY KEY,
    name VARCHAR(50) NOT NULL,
    name_en VARCHAR(50),
    category VARCHAR(20),         -- major/campus
    image_url TEXT,
    upright_meaning TEXT,         -- 正位含义
    reversed_meaning TEXT,        -- 逆位含义
    campus_context TEXT,          -- 校园场景解读
    keywords TEXT[]               -- 关键词标签
);
```

## 综合评分与优先级建议

### 功能价值评估

| 评估维度 | 评分 | 说明 |
|----------|------|------|
| **用户需求强度** | ⭐⭐⭐⭐ | 大学生对玄学/心理暗示有强烈兴趣，尤其考试季 |
| **差异化程度** | ⭐⭐⭐⭐⭐ | 市面塔罗App无校园专属，AI+校园场景是独创 |
| **开发成本** | ⭐⭐⭐ | 中等，主要工作量在视觉设计和Prompt调优 |
| **传播潜力** | ⭐⭐⭐⭐⭐ | 天然社交货币，分享意愿强 |
| **商业化潜力** | ⭐⭐⭐ | 可作为会员增值功能，但不建议前期收费 |
| **与平台调性契合度** | ⭐⭐⭐⭐⭐ | 暗黑系视觉、星火意象、AI核心，完美契合 |

### 开发优先级建议

**推荐优先级：第二阶段末期或第三阶段初期（4-5月）**

理由：
1. 功能独立性强，不依赖其他模块先完成
2. 开发成本可控，2-3周可完成MVP
3. 社交传播特性强，适合作为用户增长功能
4. 可与期末季（6月）形成时间契合

### 风险与规避

| 风险 | 规避方案 |
|------|----------|
| 被认为"封建迷信" | 强调"娱乐占卜"定位，解读末尾加免责声明 |
| AI解读质量不稳定 | 建立Prompt模板库，设置输出格式约束 |
| 用户隐私担忧 | 问题本地可删除，默认不同步云端 |
| 视觉设计投入大 | 先用开源塔罗图+滤镜处理，MVP后再定制 |

| 指标 | MVP目标 | 成熟期目标 |
|------|---------|-----------|
| 日活抽牌率 | 20%的DAU | 40%的DAU |

## 附录：视觉设计方向建议

### 整体风格

- **色调**：深紫+星火橙+银白，与平台暗黑系一致
- **材质**：玻璃拟态卡片、光粒子效果、星空背景

- 不要传统塔罗的宗教感，要现代插画风格
- 人物形象偏年轻化、校园化
- 加入星火元素（火焰、星星、光芒）
- 每张牌有专属配色，但整体和谐
