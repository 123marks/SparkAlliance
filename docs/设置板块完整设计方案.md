# 星火联盟 - 设置板块完整设计方案

## 一、项目深度分析与设计理念

### 1.1 项目现状分析

**当前项目功能模块：**
```
核心功能：
1. AI助手（Chat） - 多模型集成
2. 校园墙（CampusWall） - 社区动态
3. 智能日程（Schedule） - 日程管理
4. 星火资讯（News） - 资讯浏览
5. 星火人才（Talent） - 人才展示
6. 选课助手（Courses） - 选课辅助
7. 校园购物（Shop） - 二手交易
8. 健康打卡（Health） - 健康管理
9. 学习资源（Resources） - 资源共享
10. 学长推荐（Mentors） - 导师咨询
11. 星火规划（Planner） - 目标规划
12. 星火伴侣（Companion） - 社交功能
13. 星火共创（CoCreate） - 协作功能
14. 星火自习室（StudyRoom） - 学习空间
15. 星火消息（SparkMessages） - 消息中心
16. 星火塔罗（Tarot） - 娱乐功能
```

**当前设置现状：**
- Profile.vue中只有"偏好设置"占位符
- 缺少完整的设置体系
- 用户无法个性化定制体验
- 缺少安全与隐私设置
- 缺少通知管理
- 缺少主题与外观设置

### 1.2 设计理念

**核心理念：**
1. **用户中心**：一切设置围绕用户需求，提升用户体验
2. **简洁易用**：设置分类清晰，操作简单直观
3. **个性化**：支持深度个性化定制，满足不同用户需求
4. **安全第一**：完善的安全与隐私设置，保护用户数据
5. **智能推荐**：基于用户行为的智能设置推荐

**设计原则：**
- **渐进式披露**：常用设置优先展示，高级设置折叠隐藏
- **即时反馈**：设置修改后立即生效，提供视觉反馈
- **可撤销性**：所有设置修改可撤销，提供恢复机制
- **数据驱动**：基于用户行为数据优化设置项

## 三、核心功能模块详细设计

### 3.1 个人资料设置

#### 3.1.1 基本信息

```
设置项：
├── 用户名/昵称
│   - 当前值：显示当前用户名
│   - 修改：点击后弹出编辑框
│   - 规则：2-20字符，支持中英文、数字、下划线
│   - 实时验证：输入时实时验证格式
│   - 保存：修改后自动保存或手动保存
│
├── 真实姓名（选填）
│   - 用途：学生认证、交易信任
│   - 隐私：不公开显示
│   - 修改：需要验证身份
│
├── 性别
│   - 选项：男、女、其他、保密
│   - 用途：个性化推荐、社交匹配
│   - 隐私：可设置是否公开
│
├── 生日
│   - 格式：YYYY-MM-DD
│   - 用途：生日提醒、年龄推荐
│   - 隐私：可设置是否公开
│
├── 地区
│   - 格式：省-市-区
│   - 用途：同城推荐、线下活动
│   - 选择：三级联动选择器
│
└── 个人简介
    - 长度：0-200字符
    - 支持：文本、Emoji
    - 用途：个人主页展示
```

#### 3.1.2 头像设置

```
功能：
├── 上传头像
│   - 方式：点击上传、拖拽上传、粘贴上传
│   - 格式：JPG、PNG、GIF、WebP
│   - 大小：最大5MB
│   - 处理：自动裁剪、压缩
│
├── 裁剪编辑
│   - 工具：裁剪框、旋转、翻转
│   - 比例：1:1（圆形头像）
│   - 预览：实时预览效果
│
├── 预设头像
│   - 类型：系统预设头像库
│   - 风格：卡通、抽象、渐变
│   - 选择：点击即可应用
│
└── AI生成头像（创意功能）
    - 功能：基于用户信息AI生成个性化头像
    - 风格：多种风格可选
    - 成本：消耗AI额度
```

#### 3.1.3 教育信息

```
设置项：
├── 学校
│   - 来源：从全国高校列表选择
│   - 搜索：支持关键词搜索
│   - 验证：可选学生认证
│
├── 院系
│   - 来源：基于学校动态加载
│   - 自定义：支持手动输入
│
├── 专业
│   - 来源：从教育部专业目录选择
│   - 搜索：支持关键词搜索
│
├── 年级
│   - 选项：大一~博五、已毕业、教职工
│   - 自动更新：每年自动升级（可选）
│
├── 学号（选填）
│   - 用途：学生认证
│   - 隐私：不公开显示
│   - 验证：格式验证
│
└── 入学年份
    - 格式：YYYY
    - 用途：自动计算年级
```

### 3.2 账号与安全

#### 3.2.1 账号信息

```
显示信息：
├── 用户ID
│   - 显示：完整UUID或短ID
│   - 用途：唯一标识
│   - 操作：复制、生成短ID
│
├── 注册邮箱
│   - 显示：脱敏显示（u***@example.com）
│   - 状态：已验证/未验证
│   - 操作：修改邮箱（需验证）
│
├── 绑定手机
│   - 显示：脱敏显示（138****5678）
│   - 状态：已绑定/未绑定
│   - 操作：绑定、换绑、解绑
│
├── 注册时间
│   - 显示：YYYY-MM-DD HH:mm:ss
│   - 用途：账号年龄展示
│
└── 账号状态
    - 显示：正常、未激活、封禁
    - 说明：状态说明文字
```

#### 3.2.2 密码管理

```
功能：
├── 修改密码
│   - 流程：
│     1. 输入当前密码
│     2. 输入新密码（强度验证）
│     3. 确认新密码
│     4. 验证身份（邮箱/手机验证码）
│     5. 修改成功
│   - 安全：
│     * 密码强度提示
│     * 不能与最近5次密码相同
│     * 修改后强制重新登录
│
├── 密码强度
│   - 显示：弱、中、强
│   - 建议：提升密码强度的建议
│
└── 找回密码
    - 方式：邮箱验证、手机验证
    - 流程：验证身份 → 设置新密码
```

#### 3.2.3 绑定管理

```
绑定项：
├── 邮箱绑定
│   - 状态：已绑定/未绑定
│   - 操作：绑定、换绑、解绑
│   - 流程：验证新邮箱 → 确认绑定
│
├── 手机绑定
│   - 状态：已绑定/未绑定
│   - 操作：绑定、换绑、解绑
│   - 流程：验证新手机 → 确认绑定
│
├── 第三方账号绑定
│   - 微信：扫码绑定
│   - QQ：授权绑定
│   - GitHub：OAuth绑定
│   - 好处：快捷登录、数据同步
│
└── 学生认证
    - 状态：未认证、审核中、已认证、认证失败
    - 方式：上传学生证、学信网验证
    - 好处：解锁更多功能、提升信用
```

#### 3.2.4 登录设备管理

```
功能：
├── 当前设备
│   - 显示：设备名称、位置、最后活跃时间
│   - 标记：当前设备标识
│
├── 其他设备
│   - 列表：所有登录设备
│   - 信息：设备名称、位置、登录时间
│   - 操作：移除设备（强制下线）
│
└── 设备信任
    - 选项：信任此设备（30天内免验证）
    - 管理：查看所有信任设备
```

#### 3.2.5 安全日志

```
记录内容：
├── 登录日志
│   - 时间、IP、位置、设备、状态
│   - 异常登录提醒
│
├── 操作日志
│   - 敏感操作记录
│   - 如：修改密码、绑定手机等
│
└── 安全事件
    - 异常登录、密码尝试、账号风险
    - 提醒与处理建议
```

```
│   - 影响：谁可以查看你的个人主页

│
├── 最后活跃时间
│   - 选项：显示、隐藏
│   - 影响：是否显示最后活跃时间
│
├── 位置信息
```

#### 3.3.2 黑名单管理

```
功能：
├── 黑名单列表
│   - 显示：用户头像、用户名、拉黑时间
│   - 操作：移出黑名单
│
├── 添加黑名单

└── 举报记录
```

#### 3.3.3 数据授权

```
授权项：

│   - 选项：允许使用对话数据优化AI、不允许
│   - 说明：数据仅用于模型优化，不会泄露
│
├── 个性化推荐
│   - 选项：允许使用行为数据推荐、不允许
│   - 说明：基于你的行为推荐相关内容
│
├── 数据分析
│   - 选项：允许匿名数据分析、不允许
│   - 说明：用于改进产品体验

└── 第三方授权
```

#### 3.3.4 数据导出

```
功能：

│   - 流程：申请 → 打包 → 下载
```
