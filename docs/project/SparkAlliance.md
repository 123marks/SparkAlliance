## 一、项目核心定位

### 1.1 项目概述

**项目名称**: 星火校园 (Spark Alliance)  
**核心定位**: 以AI为特色入口，深度融合社交、生活服务与个人管理的校园超级App  
**目标用户**: 大学生 → 青年群体（学生、职场人士、自由职业者等）  
**发展路径**: 校园版 → 社区版 → 国外版

### 1.2 商业模式

- **前期**: 免费AI和工具功能吸引用户（引流）
- **中期**: 二手交易撮合费和增值服务（初步盈利）
- **后期**: B端商家、会员体系、高利润商品（规模化盈利）

### 3.5 星火人才双向匹配（Spark Talent）

**核心定位：** 颠覆传统招聘模式，从「企业发布→学生投递」的单向被动模式，转变为「青年展示能力→企业主动寻访」的双向主动模式。平台青年群体高度集中，天然形成高质量人才池。

**核心逻辑：双向可见，主动权在你**

传统模式：企业发岗位 → 学生被动投简历（低效、信息不对称）
Spark模式：青年/企业均可主动发起 → AI智能双向匹配 → 高效精准连接

**青年侧功能（能力名片）：**
- 创建「能力名片」（区别于传统简历，更立体、更有温度）
  - 擅长领域标签：编程/设计/运营/写作/数据分析/视频剪辑等
  - 作品集展示：图片/项目链接/视频演示/GitHub仓库
  - 项目经历：可直接挂载校园墙发布的项目动态
  - 可接受合作类型：实习 / 兼职 / 项目外包 / 全职
  - 期望薪资范围（可选填）
  - 所在城市/院校/专业
  - 可见性控制：公开 / 仅企业可见 / 仅好友可见
- AI能力标签自动提取与推荐
- 接收并管理企业邀请（接受/拒绝/稍后）
- 全流程投递状态追踪
- 匿名保护选项（隐藏真实姓名，企业感兴趣后再解锁）

**企业侧功能（需认证）：**
- 企业认证入驻：营业执照/官方邮箱/工商信息三重验证，防钓鱼防欺诈
- 发布岗位需求（学生可主动投递，双向通道）
- 主动搜索人才库：按技能标签/院校/城市/合作类型/匹配度筛选
- AI智能推荐候选人（语义匹配度打分）
- 向目标青年发送邀约（频率限制防骚扰）
- 候选人沟通进度管理（看板式流程追踪）

**AI匹配机制：**
- 语义向量匹配：企业JD描述 ↔ 青年能力名片（cosine相似度）
- 匹配度分值可视化展示（0-100%）
- 多维度权重：技能匹配70% + 院校背景15% + 活跃度15%
- 基于历史行为的推荐持续优化（协同过滤）
- 黑名单/屏蔽机制保护青年免受骚扰

**安全与合规：**
- 企业认证后才可主动联系，未认证仅可发布岗位
- 单日联系频率上限（防骚扰）
- 举报投诉快速响应通道
- 可选接入学信网真实信息核验
- 所有沟通记录留存，纠纷可追溯

**商业模式：**
- 青年侧永久免费，是核心资产不是收费对象
- 企业侧：基础版免费（每月限10次联系）/ 专业版付费（无限联系+AI推荐+数据报告）/ 旗舰版（置顶展示+专属客服+校园宣讲联动）
- 猎头/HR服务平台撮合佣金（未来）

**开发优先级：** 第三阶段（5-6月），企业认证模块可提前至第二阶段预研

### 3.6 星火资讯（热点爬虫聚合）

**核心定位：** 打破信息茧房，让校园青年不脱离社会脉搏。通过爬虫聚合全网热点，结合用户兴趣实现个性化推送，并提供深度分类筛选与AI辅助理解。

**信息源策略（爬虫覆盖范围）：**
- 新闻媒体：新华社、人民日报、澎湃新闻、36氪、虎嗅
- 社交平台热榜：微博热搜、知乎热榜、B站热门、抖音热点
- 垂直领域：少数派（科技）、雪球（财经）、天眼查（商业动态）
- 官方渠道：教育部通知、地方政府公告（与校园场景深度结合）
- 国际资讯：BBC中文、路透社中文（拓展国际视野）
- 就业市场：BOSS直聘趋势、智联招聘薪资报告（对接Spark Talent）

**爬虫架构：**
- 调度层：定时任务（热榜每15分钟抓取，深度内容每小时更新）
- 去重层：URL指纹哈希 + 内容语义向量去重（防止同一事件多源重复）
- 清洗层：正文抽取（Readability算法）、广告过滤、敏感内容过滤
- 存储层：原文快照（防链接失效）+ 结构化摘要入库
- 合规层：遵守各站点 robots.txt，保留原文链接归属，不侵权存储

**前端功能模块：**

①「今日热榜」- 综合热榜首屏
- 多源融合排序（热度指数综合计算）
- 资讯卡片：标题 + 来源媒体 + 热度数值 + AI生成50字摘要 + 情感标签（正面/中性/争议）
- 实时热度趋势标注：上升↑ / 下降↓ / 新入榜🔥 / 持续发酵💥

②「分类频道」- 多维度筛选体系
- 一级分类：科技与数字 / 财经商业 / 校园学术 / 社会民生 / 娱乐文化 / 国际视野 / 本地资讯
- 二级标签叠加筛选（如：科技 + AI方向 + 今日）
- 时间范围：今日 / 本周 / 自定义区间
- 热度筛选：全部 / 热门（1万+）/ 爆款（10万+）

③「为你推荐」- 个性化算法流
- 用户主动设置兴趣标签（最多10个，首次登录引导配置）
- 隐式兴趣识别：基于阅读时长/收藏/分享行为动态调整权重
- 「不感兴趣」一键标记：单条屏蔽 + 对应类别降权
- 信息茧房破防机制：单一类别连续阅读超过20条后，自动插入其他维度优质内容

④「深度阅读」- 单文章详情页
- 原文快照展示（不依赖外链，随时可读）
- AI一键摘要：3句话核心观点提炼
- AI多角度解读：正方观点 / 反方观点 / 中立分析（帮助批判性思考）
- 关联联动入口：跳转相关校园墙讨论 / 一键向AI伴侣提问该话题
- 相关资讯推荐（同事件其他报道、延伸阅读）

⑤「社交联动」- 与平台其他模块打通
- 一键分享到校园墙（附加个人评论，形成讨论）
- 收藏入个人知识库（自动按分类归档）
- 资讯下方轻量评论区（校园圈内讨论，区别于公域评论）
- 与Spark Talent联动：财经/就业类资讯下方展示相关岗位推荐

⑥「每日简报」- 主动推送
- 「今日必看」定时推送（默认每天08:00，推送5条精选）
- 用户可自定义推送时间和每日数量
- 关键词订阅：如"人工智能"、"考研政策"、"字节跳动"等，命中即推送
- 周报/月报：AI生成本周热点回顾与趋势分析

**内容安全体系：**
- 三级过滤：爬取时（URL黑名单）→ 入库前（敏感词模型）→ 展示前（实时审核）
- 敏感词过滤库动态更新（与监管合规对齐）
- 用户举报机制：举报后24小时内人工复核
- 争议内容标注「多方观点」标签，不做价值判断，呈现多元视角
- 涉政内容从严处理，宁缺勿滥

**技术方案：**
- 爬虫框架：Python + Scrapy（静态页）/ Playwright（动态渲染页）
- 任务调度：Celery + Redis（分布式任务队列）
- 存储：PostgreSQL（结构化数据）+ Elasticsearch（全文检索与相似推荐）
- AI摘要与分析：接入 DeepSeek API
- 向量存储：pgvector（语义去重与推荐）
- 缓存：Redis（热榜TTL 15分钟，用户Feed TTL 5分钟）
- 部署：作为独立微服务，与主后端解耦

**商业模式：**
- 基础资讯流：永久免费（核心留存手段）
- 付费功能：无限历史存档 / 专项深度报告 / 关键词监控预警（付费会员）
- B端服务：为企业提供行业舆情监控订阅（与Spark Talent企业账户打通）

**开发优先级：** 第三阶段（5-6月），爬虫服务作为独立微服务优先搭建，前端页面可在第四阶段精细化

## 四、技术选型

### 4.1 数据库方案

**决策：**
- **前期**: 使用Supabase
- **后期**: 可迁移到其他数据库

**原因：**
- Supabase提供完整后端服务
- 快速开发，降低初期成本
- PostgreSQL兼容性好

**注意事项：**
- ⚠️ 使用标准SQL，避免Supabase特有功能
- ⚠️ 预留数据导出接口
