### 1.2 产品演示策略

Linear使用 **自动播放的GIF** 展示核心功能:

```
特点:
├── 循环播放，无需点击
├── 展示真实使用场景
├── 文件大小控制在 2-3MB
├── 首帧即可传达核心信息
└── 静音设计（无干扰）
```

**Linear的演示GIF结构**:
1. 创建Issue (3秒)
2. 分配给团队成员 (2秒)
3. 状态变更 (2秒)
4. 循环

**星火联盟可借鉴**:
```
AI助手演示GIF (建议时长: 8-10秒)
├── 第1-2秒: 用户输入问题
├── 第2-5秒: AI思考动画 + 分步输出
├── 第5-7秒: 展示追问/继续对话
└── 第7-8秒: 展示其他功能入口
```

### 1.4 转化漏斗设计

```
Linear的转化路径:

首页
  ↓ (首屏CTA)
Get started
  ↓ (邮箱注册)
填写邮箱 + 密码
  ↓ (自动跳转)
创建工作空间
  ↓ (引导)
邀请团队成员
  ↓ (产品内)
创建第一个Issue ← 完成激活

路径特点:
├── 无需信用卡
├── 最小字段注册 (仅需邮箱)
├── 即时产品体验
└── 逐步引导，不 overwhelm
```

**星火联盟可借鉴**:
```
优化后的注册路径:

首页 "极速启航"
  ↓
极简注册 (邮箱/手机号 + 密码)
  ↓
选择身份 (学生/职场/其他)
  ↓
可选: 填写学校/专业
  ↓
进入App首页
  ↓ (引导浮层)
"试试和AI打个招呼吧"
  ↓
用户发送第一条消息 ← 激活完成
```

```
Slogan: "Where teams design together"

├── 大胆 (Bold typography)
├── 协作 (Real-time collaboration)
├── 专业 (Professional tools)
```

- 使用非常规字体 (Whyte Inktrap)

- 不规则图形元素

**星火联盟品牌建议**:

```diff
当前品牌感觉:
```
