# 反重力启动指令

## 🚀 复制以下指令到反重力

```
请加载全局规则并开始工作。

## 第一步：加载全局规则
阅读文件：.agents/antigravity-rules.md
严格遵守所有规则。

## 第二步：语言设置
- 交互语言：中文
- 代码注释：中文
- 文档编写：中文

## 第三步：代码规范
- 代码简洁干净
- 注释完整清晰
- 变量命名有意义

## 第四步：开发前检查
每次开发前必须：
1. 阅读核心文档（PROJECT-RULES.md, SparkAlliance.md）
2. 判断是否需要加载Skills
3. 判断是否需要使用MCP
4. 确认任务要求

## 第五步：开始工作
当前项目：星火校园
当前阶段：MVP准备阶段
首要任务：前端设计

请确认已理解规则，然后等待用户指令。
```

## 🎯 开发任务指令示例

### 前端设计任务
```
任务：设计前端界面

必载Skills：/skill ui-ux-pro-max

要求：
- 玻璃拟态UI
- 弹簧阻尼动画
- 鼠标跟随效果
- 粒子动感背景

输出：frontend/src/components/

完成后：Git提交，更新AGENTS.md
```

### 核心功能开发任务
```
任务：开发用户认证系统

必载Skills：/skill systematic-debugging
             /skill test-driven-development

要求：
- 注册/登录功能
- 多群体身份选择
- JWT认证
- 单元测试

输出：backend/src/controllers/

完成后：Git提交，更新AGENTS.md
```

### Bug修复任务
```
任务：修复登录bug

必载Skills：/skill systematic-debugging

步骤：
1. 分析bug原因
2. 定位问题代码
3. 实施修复
4. 测试验证
5. Git提交

完成后：更新AGENTS.md，记录修复过程
```
